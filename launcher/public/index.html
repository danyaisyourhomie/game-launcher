Нажмите "Играть" на mbtl.ru

  • Инструкция
  • 1. Открыть лаунчер
  • 2. Залогиниться на сайте mbtl.ru
  • 3. Нажать кнопку "Играть"
  • 4. Вернуться к лаунчеру
  • Первая загрузка лаунчера может занять какое-то время

Не забудьте скачать последнюю Java и JDK (17 версия)
